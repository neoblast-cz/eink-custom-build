===========================
eink-custom-build/
├── app.py                          # Entry point + module registry
├── config.json / config.example.json
├── requirements.txt

├── .gitignore
├── core/

├── modules/
│   ├── base.py                     # BaseModule (extend this for new modules)

├── web/
│   ├── routes.py                   # All Flask routes (single file)
│   └── templates/                  # Dark-themed UI (5 templates)
├── static/style.css                # InkyPi-inspired dark theme
└── uploads/                        # Photo storage



To test locally
===========================
You'll need to install Python first (from python.org), then:


pip install -r requirements.txt
python app.py
# Open http://localhost:8080
It will run in mock mode (no e-ink hardware) and save preview images to static/preview.png.

To deploy on Pi
===========================
# SSH into the Pi, then:
git clone https://github.com/neoblast-cz/eink-custom-build.git /home/pi/einkpi
cd /home/pi/einkpi
bash install.sh
The web UI will be available at http://raspberrypi:8080. Upload photos, set an ICS calendar URL, and switch between modules from the dashboard.

http://raspberrypi:8080

Useful commands:
===========================
sudo systemctl status einkpi    # Check status
sudo systemctl restart einkpi   # Restart
sudo journalctl -u einkpi -f    # View logs

Update:
===========================
git pull
sudo systemctl restart einkpi
